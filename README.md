# nodeImg
### 主要功能
 - 利用node 爬虫 爬取并保存 多页 图片
 - 利用node socket.io 实现服务器和 浏览器通信，显示实时下载进度
 - 可浏览已下载的图片
### 主要技术
 - node  express 
 - node 插件 cheerio  request   superagent  superagent-throttle ...
